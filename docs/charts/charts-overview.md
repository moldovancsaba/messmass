## Purpose
This folder is the source of truth for how charts should be configured (Admin), rendered (Report UI), and kept visually consistent (layout/typography rules).

## Canonical Documents
- `docs/charts/charts-chart-alignment-summary.md` - KPI/Pie/Bar layout alignment architecture and testing checklist.
- `docs/charts/charts-table-chart-usage-guide.md` - How to create and populate a `table` chart (variables, formulas, template wiring).

## What To Read When
- **A chart is visually misaligned / overflowing**: start with `docs/charts/charts-chart-alignment-summary.md`.
- **You need to configure a table chart**: use `docs/charts/charts-table-chart-usage-guide.md`.

## Historical / Archived Material
Older one-off fix notes and pixel-level analyses live under `docs/archive/_archive/charts/`. Prefer the canonical docs unless you are debugging a regression tied to a specific change set.
