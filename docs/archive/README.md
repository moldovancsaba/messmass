# Archived Documentation

Everything under `docs/archive/` is **historical, non-authoritative material** kept for
reference only. It is intentionally excluded from the documentation consistency audit
(`scripts/docs-consistency-audit.js`).

Internal links within these files may be broken and are **not maintained** — they point at
paths that were valid when the document was written. For current, canonical documentation
start at [`docs/index.md`](../index.md) and [`docs/_meta/meta-canonical-map.md`](../_meta/meta-canonical-map.md).
